Générateur de Signature Email

Remplis tes informations, copie ta signature et ajoute-la à ton client mail en 2 minutes.

Installation
1

Copie ta signature

Clique sur "Copier la signature" ci-dessus.

2

Ouvre les paramètres Gmail

Sur Gmail, clique sur ⚙️ en haut à droite puis "Voir tous les paramètres".

⚙️ → Voir tous les paramètres
3

Trouve la section Signature

Dans l'onglet "Général", descends jusqu'à la section "Signature".

4

Crée et colle

Clique sur "+ Créer", nomme-la "M&E Consulting", puis colle dans la zone de texte.

Ctrl + V  ou  Cmd + V
5

Active par défaut et enregistre

Dans "Paramètres par défaut", sélectionne ta signature pour les nouveaux mails et réponses. Descends et clique "Enregistrer les modifications".

1

Copie le code HTML

Clique sur "Copier le code HTML" ci-dessus (le bouton bleu).

2

Ouvre les paramètres Outlook

Outlook Web : ⚙️ → Courrier → Rédiger et répondre.

Outlook Desktop : Fichier → Options → Courrier → Signatures.

Paramètres → Courrier → Signatures
3

Crée une nouvelle signature

Clique sur "+ Nouvelle signature" et nomme-la "M&E Consulting".

4

Colle la signature

Outlook Web : colle directement dans la zone de texte (Ctrl + V).

Outlook Desktop : dans l'éditeur, clique sur l'onglet "Source" ou "Code source", colle le code HTML, puis reviens en mode normal.

5

Définis par défaut et enregistre

Sélectionne ta signature comme défaut pour les nouveaux messages et les réponses, puis enregistre.

1

Copie le code HTML

Clique sur "Copier le code HTML" ci-dessus.

2

Crée une signature dans Mail

Ouvre Mail → Réglages (ou Préférences) → onglet "Signatures". Clique sur "+" pour en créer une nouvelle.

Mail → Réglages → Signatures → +
3

Ferme Mail

Quitte complètement l'application Mail.

4

Modifie le fichier de signature

Ouvre le Finder, va dans le menu Aller → Aller au dossier et entre :

~/Library/Mail/V10/MailData/Signatures/

Ouvre le fichier .mailsignature le plus récent avec TextEdit, remplace le contenu HTML par le code copié, puis enregistre.

5

Verrouille le fichier

Fais un clic droit sur le fichier → Lire les informations → coche "Verrouillé" pour empêcher Mail de le réécraser. Rouvre Mail.

1

Copie le code HTML

Clique sur "Copier le code HTML" ci-dessus.

2

Crée un fichier HTML

Ouvre un éditeur de texte (Bloc-notes, TextEdit), colle le code HTML et enregistre le fichier en signature.html sur ton bureau.

3

Ouvre les paramètres du compte

Dans Thunderbird : Menu (☰) → Paramètres des comptes.

☰ → Paramètres des comptes
4

Ajoute la signature

Coche "Apposer la signature à partir d'un fichier", puis sélectionne le fichier signature.html que tu as créé. Coche aussi "Utiliser le format HTML".

5

Enregistre

Clique sur OK. Ta signature apparaîtra automatiquement dans tes nouveaux emails.

Astuce : si la signature s'affiche mal après le collage, supprime d'abord tout le contenu de la zone de signature avant de coller. N'utilise pas le mode texte brut.
